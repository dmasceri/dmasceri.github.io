# dmasceri.github.io

Source for my personal project page, served via GitHub Pages at
**https://dmasceri.github.io**.

A single, self-contained `index.html` (no build step, no dependencies) that
introduces the things I build and links out to each project's repository:

- **[RAMPS](https://github.com/dmasceri/ramps)** — automated FE bridge modeling &
  load rating (doctoral research archive).
- **[MORPHFLIGHT](https://github.com/dmasceri/morphflight)** — on-rails cave flyer
  built to lock down a morph-weapon mechanic.
- **[Intraday Regime Dashboard](https://github.com/dmasceri/intraday-regime-dashboard)**
  — read-only market regime & intraday observation tool.

## Editing

Just edit `index.html` and push to `main`. GitHub Pages redeploys automatically.
To add a project, copy an existing `<article class="card">` block and update the
title, tagline, description, tags, and link.
